# Audience Intelligence Platform

Internal prototype built by Big Village.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- Anthropic Claude API

## Modules

1. Audience Builder
2. Intelligence
3. Concept Testing
4. Focus Group Studio
5. Orchestration
6. Brand Monitor
7. Survey Simulator

## Setup

```bash
npm install
npm run dev
```

Set your Anthropic API key inside the app via the key manager in the sidebar footer.
